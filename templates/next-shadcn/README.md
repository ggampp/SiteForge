# SiteForge — Next + Tailwind template

Minimal Next.js app for rebuilds driven by SiteForge extract/spec/skill.

```bash
pnpm install
pnpm dev
```

Then:

```bash
siteforge extract https://example.com
siteforge download <sourceId> -t public/images
# implement sections under app/ and components/
```

Or open the static rebuilds under `../../examples/clones/`.
